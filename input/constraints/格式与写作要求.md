# 格式与写作要求

本文件用于汇总学校模板、老师要求和本次论文的额外限制。若用户没有手动填写，agent 应在对话中主动提取并补全关键信息。

## 模板信息

- 使用模板文件：`template/南信大毕业论文规范.docx`
- 模板是否已核对：是
- 是否已有明确封面要求：是，使用南信大本科毕业论文统一封面
- 是否需要目录：是，按三级标题编排

## 标题与层级要求

- 论文题目格式：中文题目黑体三号居中；题目一般不宜超过 20 字
- 一级标题格式：黑体四号顶格
- 二级标题格式：黑体小四号顶格
- 三级标题格式：宋体小四号顶格

## 正文格式要求

- 正文字体：中文宋体，英文 Times New Roman
- 正文字号：正文小四号
- 行距：1.5 倍行距
- 段前段后：章标题段前 0.8 行、段后 0.5 行；节标题段前 0.5 行、段后 0.5 行
- 页边距：A4；有页眉时上 2.1cm、下左右 2.5cm；无页眉时四边均 2.5cm
- 页码要求：摘要和目录用罗马数字；正文起用阿拉伯数字并底部居中

## 摘要与关键词要求

- 中文摘要要求：需要，约 150-350 字，概括研究目的、内容、方法与结论
- 中文关键词数量：3-5 个
- 英文摘要要求：需要，与中文摘要内容一致

## 参考文献要求

- 参考文献格式标准：GB/T 7714-2015
- 是否要求知网格式优先：是
- 是否要求文献数量下限：模板未明确
- 是否要求近年文献比例：近五年文献不少于总文献数量的 1/3；外文文献原则上不少于总文献数量的 1/5

## 内容写作要求

- 是否偏综述：是
- 是否强调学术风：是
- 是否允许明显口语化表达：否
- 是否需要避免绝对化结论：是

## 当前补充判断

- 当前模板是本科毕业论文规范，要求严于一般课程论文。
- 若老师对课程论文允许简化，可后续确认是否保留英文摘要、致谢、声明等模块。
- 在未获得老师简化说明前，正文、大纲和参考文献先按较完整学术论文结构准备。

## agent 使用说明

如果本文件中有空缺，agent 应按以下顺序处理：

1. 先从模板中提取能确定的信息。
2. 再从用户对话中补充。
3. 仍无法确认时，在校验报告中标记为待确认项。

agent 不应在格式要求不明确时假装已经完全满足模板。
